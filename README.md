## Como inicializar o projeto:

Clone  o repositório: 
```
 git clone https://github.com/M3nozzi/frontend_nave.git
```
Instale os pacotes
```
 yarn install ou npm install
```

Execute o projeto:
```
yarn start ou npm start
```
Para fazer o login, utilize os dados abaixo:
```

```
